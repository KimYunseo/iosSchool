Closure
===

코드의 블럭

일급 객체

 전달인자, 변수, 상수 등으로 저장, 전달이 가능
 
함수 : 이름이 있는 클로저
 
매개변수, 반환 타입 생략가능

return 키워드 생략 가능

축약된 전달인자 이름을 사용 가능

후행 클로저

{ (매개변수들) -> 반환타입 in
	실행 코드
}

in이 쓰인 이유 : 정의부와 실행부를 분리하기 위해 , 앞에 파라메터가 없는 경우 생략가능
